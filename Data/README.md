# Race Data

This folder contains the raw FastF1-derived race CSV files used by the current study.

## Circuits

| Folder | Grand Prix | Circuit |
|---|---|---|
| `Bahrain/` | Bahrain Grand Prix | Bahrain International Circuit |
| `Saudi Arabia/` | Saudi Arabian Grand Prix | Jeddah Corniche Circuit |
| `United States/` | United States Grand Prix | Circuit of the Americas |
| `Italy/` | Italian Grand Prix | Autodromo Nazionale Monza |
| `Hungary/` | Hungarian Grand Prix | Hungaroring |

Each circuit folder keeps race laps, race weather, and race results for the 2022-2025 seasons when available. Free-practice data and circuits outside the current article scope are not part of the cleaned GitHub version.
